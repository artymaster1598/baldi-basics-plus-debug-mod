# baldi-basics-plus-debug-mod
A debug/cheat menu mod for Baldi's Basics Plus.

Features:
- Spawn, freeze, delete any NPC (including modded)
- God Mode, Infinite Stamina, Stealth, Noclip, Fly
- Collect all notebooks, solve math machines, force win
- Trigger/stop random events
- Give or drop any item (including modded)
- Teleport to rooms, bookmarks, Baldi
- Baldi anger control
- Time scale (pause/slow/fast)
- Free camera + screenshots
- Live stats (FPS, position, YTP, notebook count)
- BepInEx log viewer
- Configurable hotkeys (F1 = toggle debug, F2 = menu)
- RU/EN language support

Requirements:
- Baldi's Basics Plus (latest)
- BepInEx 5.4.23+
- MTM101BaldAPI

Installation:
Drop DebugMod.dll into BepInEx/plugins/
